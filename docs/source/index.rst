specargs
========

**specargs** is a Python library for serializing and deserializing request and response data and generating OpenAPI
Specification (OAS) files built upon :doc:`marshmallow <marshmallow:index>`, :doc:`apispec <apispec:index>`, and
:doc:`webargs <webargs:index>`.
